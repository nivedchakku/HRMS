
# HRMS
HRMS is a website for manage employees  inside a organization.

## Requirments
- Visual Studio
- MySQL Database

##  How To Use
- Clone The Repo
- Paste the project folder inside a folder
- Create database "HRMS" in MySQL 
- Open the project in VS 2022
- Build and run the project
